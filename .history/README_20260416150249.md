# ProxVanta For Cursor

This repo exists to get ProxVanta working in Cursor.

## Quick Start

```bash
bash scripts/install-cursor.sh
export PROXVANTA_MCP_TOKEN="your_org_api_key_here"
```

Restart Cursor, then try:

```text
Use ProxVanta to list the available Agent Contexts.
```

## Files

- `scripts/install-cursor.sh`: installs ProxVanta into `~/.cursor/mcp.json`
- `plugins/proxvanta`: standalone Cursor plugin package
