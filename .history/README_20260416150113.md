# ProxVanta Cursor Plugins

Standalone Cursor plugin package for ProxVanta.

## Fastest Start

Run:

```bash
bash scripts/install-cursor.sh
```

Then export your organization API key and restart Cursor:

```bash
export PROXVANTA_MCP_TOKEN="your_org_api_key_here"
```

This installer writes `~/.cursor/mcp.json` with the production ProxVanta MCP server and `bearerTokenEnvVar`.

OAuth is not required for the shortest working setup here. The current hosted Cursor path in this repo uses the environment-variable bearer token flow documented in the main ProxVanta MCP setup docs.

## Package

- `plugins/proxvanta`: production

This package contains:

- `.cursor-plugin/plugin.json`
- `mcp.json`
- `skills/proxvanta/SKILL.md`
- `README.md`

## Auth

ProxVanta currently authenticates hosted MCP with an organization API key.

The packaged Cursor config uses:

- `Server URL`: `https://api.proxvanta.com/mcp`
- `Authorization`: bearer token from `PROXVANTA_MCP_TOKEN`

Before using the plugin in Cursor, export:

```bash
export PROXVANTA_MCP_TOKEN="your_org_api_key_here"
```

Then restart Cursor so it can pick up the environment variable.

## Install Script

The repo includes:

- `scripts/install-cursor.sh`

It creates `~/.cursor/mcp.json` if needed, then adds or updates a `proxvanta` MCP server entry that points to production and uses `PROXVANTA_MCP_TOKEN` for auth.

## Endpoint

- Production: `https://api.proxvanta.com/mcp`
